Basicamente, a instalação desta VM foi feita da seguinte forma:

- SO: Lubuntu 18.04 (32bits)

- Comandos após a instalação:
	sudo apt-get update
	sudo apt-get install gcc
	sudo apt-get install flex
	sudo apt-get install bison
	sudo apt-get install default-jre

- Jasmin:
	http://jasmin.sourceforge.net/
	Versão em uso: 2.4
	Teste: 
		cd /home/com0002/Desktop/jasmin
		java -jar jasmin.jar examples/HelloWorld.j
